## 🚀 Sobre el Proyecto
El **Centro de Innovación y Productividad** es la principal plataforma de inteligencia territorial y académica desarrollada por Cultura T. Su misión es democratizar el análisis científico de datos, ofrecer capacitación asincrónica especializada (Academia SGR) y realizar auditorías exhaustivas sobre las inversiones y el desarrollo en los territorios.

No es solo una web; es un ecosistema operativo de auditoría cívica y formación estratégica en turismo, sostenibilidad y economía regional. Para conocer más sobre nuestra labor corporativa y proyectos a gran escala en todo el país, **[Visita nuestro sitio web oficial](https://cultura-t.com)**.

## 🌍 Compromiso Open Data y Transparencia
En Cultura T creemos en la **Apropiación Social del Conocimiento**. 
Gran parte de la arquitectura visual de visualización (incluyendo los *Dashboards* interactivos y la interfaz React que ves en este repositorio) está estructurada bajo una filosofía algorítmica transparente. Operamos sobre registros oficiales del DANE, Banco Mundial, CEPAL y FONTUR cruzándolos a través de ecosistemas de **Open Data**. Nuestro propósito es que las lógicas de democratización de la información estén siempre disponibles para fortalecer a investigadores y administradores públicos.

## 🛠️ Arquitectura Tecnológica
Este proyecto está desplegado nativamente utilizando infraestructura de alto rendimiento y arquitectura Serverless Data:
- **Core Engine:** React 19 + Vite (Rendimiento CDN Ultrarrápido).
- **Data Visualization:** Recharts + D3-Geo para interfaces "Scrollytelling".
- **Backend & Modelado (Desacoplado):** Modelos Bayesianos de Markov Chain Monte Carlo (MCMC) procesados en Python (cuya lógica subyacente forma nuestra IP) exportados como nodos estáticos JSON altamente seguros y eficientes.
- **Seguridad SEO:** Optimizado a nivel de metadatos profundos con React Helmet Async y Schema JSON-LD.

*(Nota de Seguridad: Los repositorios de datos locales crudos, cuadernos Py y documentos fiscales operan bajo un esquema de privación estricta y están bloqueados en el repositorio público mediante .gitignore para salvaguardar la intimidad corporativa de la firma y sus clientes gubernamentales).*
