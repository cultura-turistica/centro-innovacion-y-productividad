# Guía de Estilo Maestro (IA / Gráfica)
**Cultura T - Inteligencia Artificial SGR**

Esta guía resguarda el prompt exacto instruido para mantener homogeneidad visual en todo el proyecto de ASC.

## El Prompt Maestro

Cualquier futura generación de imagen (avatares, paisajes, esquemas) en este sistema debe inyectarse con este sufijo restrictivo:

> "Ilustración naturalista y científica. Trazo de tinta detallado con texturas de acuarela vibrante. Inspirado en fichas técnicas botánicas y de fauna (estilo USAID/Procolombia). Estilo de Personajes y objetos: 3D amigable y orgánico, con texturas de "arcilla" o "pintura fresca". Deben ser expresivos y coloridos, alejados del estilo corporativo plano (Flat Art). Línea Gráfica: "Desenfadada" y "Hecha a mano". Usa bordes imperfectos, elementos que parezcan bocetos rápidos pero profesionales. Base blanca. CRITICAL REQUIREMENT: ABSOLUTELY NO TEXT, NO LOGOS, NO LETTERS, NO TYPOGRAPHY, NO STAMPS."

## Elementos Centrales
1. **Colores Soportados:** Azul Catalina, Verde Amazon, Naranja Clementine.
2. **Atmósfera:** Botánica, Naturalista, Académica de campo (Libreta verde / Ecológica).
3. **Restricciones:** Jamás renderizar símbolos alfabéticos o logos para evitar alucinaciones visuales de la red neuronal.
